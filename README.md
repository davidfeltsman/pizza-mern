# pizza-mern
Admin panel for react pizza
***
Регистрация, редактирование товаров, управление пользователями, управление заказами

Node version 14.15.0

Client stack:
```
react
redux
redux-thunk
axios
material ui
react-hook-form
```
Server:
```
nodejs
express
mongodb
mongoose
passport
jwt
express-validator
nodemailer
```
